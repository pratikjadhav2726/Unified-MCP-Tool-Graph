# Experimental Features

This directory contains experimental and work-in-progress features that are not yet ready for production use.

## ⚠️ Warning

**These features are experimental and may:**
- Have incomplete functionality
- Contain bugs
- Change significantly or be removed
- Not be fully documented
- Require additional setup or dependencies

Use at your own risk!

## Current Experimental Features

### `mcp_adder_to_ide.py`

Experimental script for adding MCP servers to IDE configurations (e.g., Cursor, VS Code).

**Status:** Work in progress

**Usage:** Not recommended for production use. See main documentation for stable alternatives.

---

**Note:** Experimental features may be moved to the main codebase, archived, or removed in future releases.
